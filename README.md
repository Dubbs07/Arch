# Arch
Arch Dotfiles

Anything prefixed with "hypr" is relating to overall hyprland configurations
All hyprland files sit at /home/%yourusername%/.config/hypr
Wallpaper script randomizer script sits at /home/%yourusername%/.config/hypr/scripts
Wallpaper images sit at /home/%yourusername%/.config/hypr/wallpapers

Anything NOT prefixed with "hypr" is waybar configurations
All Waybar configuration sits at /home/%yourusername%/.config/waybar
Launch script sits at /home/%yourusername%/.config/waybar/scripts
